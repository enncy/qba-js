# qba

题库解析器，将题库文本解析成表格或者可处理的数据

## 使用方法

```bash
# 下载依赖
npm install qba-js
```

```js
const qba = require('qba-js');
const content = `...题库文本...`;
console.log(qba.parse(content));
```
